# 控制台学生管理系统 — 需求文档

## 项目定位

一个纯 Java 控制台程序，不依赖数据库和外部框架，适合 Java 初学者练手。

## 技术约束

- JDK 版本：8 或以上
- 数据存储：内存（`ArrayList`），程序退出即丢失
- 交互方式：`System.in` / `System.out`，菜单驱动
- 不依赖任何第三方库

---

## 功能需求

### 1. 主菜单

程序启动后显示主菜单，等待用户输入：

```
========== 学生管理系统 ==========
1. 添加学生
2. 删除学生
3. 修改学生
4. 查询学生
5. 显示所有学生
0. 退出系统
===================================
请选择操作：
```

- 输入非法选项时提示"输入有误，请重新选择"并重新显示菜单
- 选择 `0` 时输出"感谢使用，再见！"并退出程序

### 2. 添加学生

选择 `1` 后：

1. 依次提示用户输入以下信息：
   - 学号（字符串，如 "S001"）
   - 姓名（字符串）
   - 年龄（整数）
   - 性别（字符串，男/女）
   - 专业（字符串）

2. 校验规则：
   - 学号不能为空，且不能与已有学生重复 → 提示"学号已存在，添加失败"
   - 姓名不能为空
   - 年龄必须在 1 ~ 150 之间 → 提示"年龄不合法"
   - 性别必须是"男"或"女"
   - 专业不能为空

3. 校验通过后，将学生信息存入集合，输出"添加成功！"

### 3. 删除学生

选择 `2` 后：

1. 提示输入要删除的学号
2. 根据学号查找学生：
   - 找到 → 显示学生信息，二次确认"确认删除？(y/n)"
     - 输入 `y` 或 `Y` → 删除，输出"删除成功！"
     - 输入 `n` 或 `N` → 取消删除，输出"已取消删除"
   - 找不到 → 输出"未找到该学生"

### 4. 修改学生

选择 `3` 后：

1. 提示输入要修改的学号
2. 根据学号查找学生：
   - 找到 → 显示当前信息，然后逐字段提示输入新值（直接回车表示不修改该字段）
     - 姓名：回车跳过，否则更新
     - 年龄：回车跳过，否则校验 1~150
     - 性别：回车跳过，否则校验"男"/"女"
     - 专业：回车跳过，否则更新
   - 修改完成后输出"修改成功！"
   - 找不到 → 输出"未找到该学生"

### 5. 查询学生

选择 `4` 后，进入一个子菜单：

```
---------- 查询学生 ----------
1. 按学号查询
2. 按姓名查询
3. 返回主菜单
-------------------------------
请选择：
```

#### 5.1 按学号查询
1. 输入学号
2. 找到 → 显示学生详情
3. 找不到 → 输出"未找到该学生"

#### 5.2 按姓名查询
1. 输入姓名（支持模糊匹配，包含即可）
2. 显示所有匹配的学生列表
3. 无匹配 → 输出"未找到匹配的学生"

#### 5.3 返回主菜单
回到主菜单

### 6. 显示所有学生

选择 `5` 后：

1. 如果没有任何学生 → 输出"暂无学生数据"
2. 否则以表格形式显示所有学生信息：

```
学号      姓名      年龄      性别      专业
--------------------------------------------------
S001      张三      20        男        计算机科学
S002      李四      21        女        软件工程
--------------------------------------------------
共 2 名学生
```

## 二期需求：用户认证模块

### 概述

在现有学生管理系统上增加用户注册、登录、验证码、忘记密码功能。程序启动后先进入认证菜单，登录成功后才能进入学生管理主菜单。

### 程序启动流程

```
========== 学生管理系统 ==========
1. 登录
2. 注册
3. 忘记密码
0. 退出系统
==================================
请选择操作：
```

- 未登录时无法进入学生管理功能
- 登录成功后进入原有主菜单（添加/删除/修改/查询/显示学生）

---

### 1. 注册

选择 `2` 后：

1. 依次提示用户输入：
   - 用户名（字符串，不能为空，不能与已有用户重复,字母加数字不能是纯数字，）
   - 密码（字符串，不能为空）
   - 身份证号（字符串，18 位，前面十七位只能是数字最后一位可以是X）
   - 手机号（字符串，11 位）

2. 校验规则：
   - 用户名不能为空 → 提示"用户名不能为空"
   - 用户名不能重复 → 提示"用户名已存在"
   - 密码不能为空 → 提示"密码不能为空"
   - 身份证号必须 18 位 → 提示"身份证号格式错误"
   - 手机号必须 11 位 → 提示"手机号格式错误"

3. 全部校验通过后写入用户集合，输出"注册成功！"

---

### 2. 登录（含验证码）

选择 `1` 后：

1. 输入用户名
   - 查找用户，找不到 → 提示"用户不存在"

2. 输入密码
   - 密码错误 → 提示"密码错误"

3. 验证码校验：
   - 系统生成随机 **4 位数字** 验证码并显示在屏幕上，如：`验证码：3847`
   - 提示用户输入验证码
   - 输入错误 → 提示"验证码错误"，重新生成验证码，重新输入
   - 验证码正确 → 登录成功，输出"登录成功，欢迎 {用户名}！"

4. 登录成功后进入原有学生管理主菜单

---

### 3. 忘记密码

选择 `3` 后：

1. 输入用户名
   - 查找用户，找不到 → 提示"用户不存在"

2. 身份验证（两项都要匹配）：
   - 输入身份证号 → 与注册时填写的身份证号比对，不匹配 → 提示"身份验证失败"
   - 输入手机号 → 与注册时填写的手机号比对，不匹配 → 提示"身份验证失败"

3. 两项都匹配后：
   - 提示输入新密码
   - 新密码不能为空 → 提示"密码不能为空"
   - 更新密码，输出"密码重置成功，请重新登录！"
